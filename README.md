# ZAKOTrackerV2 - Hardware

![TopCover](./pic/TopCover.png)

自作VRモーショントラッカー「ZAKOTracker」V2の基板設計データ。

BNO055 IMUの値をESP-NOWで送信するセンサーノード基板です。

## 関連リポジトリ

- [ZAKOTrackerV2_ESP8266](https://github.com/Tahi314/ZAKOTrackerV2_ESP8266) - 送信側ファームウェア（ESP8266）
- [ZAKOTrackerV2_ESP32](https://github.com/Tahi314/ZAKOTrackerV2_ESP32) - 受信側ファームウェア（ESP32）

## 設計環境

- KiCad 6.0

## ディレクトリ構成

- `kicad/` - KiCad プロジェクトファイル
- `3d/` - 3Dモデル
- `pic/` - 画像
